name : OJAS NAIK
mis : 111503048
This is a game made to improve the players typing speed.
Random words move across the screen and the player has to type them correctly before the word reaches the end of screen.
There are three levels a the speed varies acordingly.
Also , later in the game the amount of words and speed also increases.
Game is over once the player misses 10 words.
Full screen is preferred.
libraries used : ncurses and menu.
